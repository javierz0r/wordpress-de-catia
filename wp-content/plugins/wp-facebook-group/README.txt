== Description ==

*WP FACEBOOK GROUP PROMOTER: THE 1ST FACEBOOK GROUP PROMOTION & INVITATION PLUGIN* 

Are you looking to **add facebook join group button** on your wordpress powered website so that you can convert a larger percentage of your website visitors to become your facebook group members? 

Are you interested in a customized way to promote your facebook group(s) via your website? 

**WP Facebook Group Promoter Plugin** is the right tool you need to promote and invite your website/blog readers to your group. 

The plugin lets people join your *Facebook group* from a link on your wordpress website posts/pages, widget or via shortcode.

This is completely customizable, responsive & search engine friendly. Download or install now to get started.

== Frequently Asked Questions ==

= How to install wordpress facebook group plugin =

This section describes how to install the plugin and get it working.

From Within Wordpress:
1. Visit "Plugins > Add New".
2. Search for "WP Facebook Group Promoter".
3. Activate "WP Facebook Group Promoter" from your Plugins page.
4. See post activation instructions below.

Manually:
1. Upload `kankoz-fg-promoter` folder to the `/wp-content/plugins/` directory
2. Activate the WP Facebook Group Promoter plugin through the 'Plugins' menu in WordPress
3. See post activation instructions below.

After Activation:
1. Check a top level menu named: "FB Group Promoter" at the left hand side of your wordpress dashboard
2. Click this menu & finalize the setup with few clicks. It's so simple.
